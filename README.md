## ğŸš€ Getting Started

### **Prerequisites**
```bash
- PHP >= 8.1
- Composer
- Node.js >= 16
- SQLite/MySQL
```

### **Installation**

1. **Clone the repository**
   ```bash
   git clone https://github.com/Lanxxxe/coil-project.git
   cd coil-app
   ```

2. **Install PHP dependencies**
   ```bash
   composer install
   ```

3. **Install JavaScript dependencies**
   ```bash
   npm install
   ```

4. **Environment setup**
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

5. **Database setup**
   ```bash
   php artisan migrate
   php artisan db:seed
   ```

6. **Build assets**
   ```bash
   npm run build
   ```

7. **Start development server**
   ```bash
   php artisan serve
   ```
